# 🐱‍👤 Pokémon Team Builder

Приложение, в котором ты можешь собирать свою команду покемонов, смотреть покемонов других пользователей и управлять своим аккаунтом. Всё просто и в стиле боевого тренера.

---

## ✨ Возможности

- 🔐 Авторизация через Google (Firebase)
- 📁 Хранение данных в Firestore
- 🖼️ Загрузка изображений через imgbb API
- 🧠 Управление состоянием через Redux Toolkit
- 🕹️ Добавление покемонов в свою команду
- 🌍 Просмотр других пользователей и их команд
- 💅 Два подхода к стилям: Tailwind CSS и Sass

---

## 🚀 Как запустить

1. Установи зависимости:

```bash
npm install
```
